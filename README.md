
# Oa-Eoms

## 简介
投诉工单自动接单及预分配

## 主要功能
- 登陆公司内部OA系统和运维系统
  - 密码登录OA，自动跳转至运维系统
- 自动受理工单
  - 读取投诉工单列表
  - 过滤当前未受理的投诉工单
  - 自动受理投诉工单

`pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple/`
